## Come Contribuire

### 1. Segnalare Bug

Se trovi un bug, per favore apri una issue includendo:
- Descrizione dettagliata del problema
- Input che causa l'errore
- Output atteso vs output ottenuto
- Sistema operativo e versione del compilatore

### 2. Proporre Miglioramenti

Per proporre nuove funzionalitÃ  o miglioramenti:
- Apri una issue con il tag `enhancement`
- Spiega il motivo della proposta
- Se possibile, fornisci esempi o pseudocodice

### 3. Inviare Pull Request

1. Fai un fork del repository
2. Crea un branch per la tua feature (`git checkout -b feature/AmazingFeature`)
3. Assicurati che il codice rispetti la Norminette
4. Testa le modifiche con `make test` e `./test.sh`
5. Commit delle modifiche (`git commit -m 'Add some AmazingFeature'`)
6. Push al branch (`git push origin feature/AmazingFeature`)
7. Apri una Pull Request
